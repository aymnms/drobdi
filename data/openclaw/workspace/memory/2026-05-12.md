- La stratégie professionnelle actuelle d’Aymerick est en deux temps : 1) retrouver un travail le plus vite possible pour regagner de l’argent, un cadre de vie et un meilleur équilibre mental ; 2) une fois stabilisé, se former sérieusement sur Rust en parallèle, notamment via des projets personnels comme `cetaitquand.fr`, puis chercher à terme un poste orienté Rust.
- Le choix actuel du backend Python est donc pragmatique et court terme : Aymerick estime qu’il y a davantage d’opportunités dans cette stack pour se remettre rapidement en selle.
- L’intérêt d’Aymerick pour Rust vient en profondeur de son goût initial pour le langage C et pour le bas niveau. Le premier vrai langage qu’il a appris et aimé est le C, découvert en terminal à travers le projet `mario-isn`, consolidé pendant ses études puis retravaillé plus récemment.
- Aymerick aime les langages bas niveau, la rigueur technique, la proximité machine et les systèmes contraints, mais n’a jamais eu d’expérience professionnelle directe dans cet univers faute d’opportunités.
- Aymerick voit Rust comme une évolution moderne et réaliste du C : il apprécie l’idée d’un langage bas niveau, rigoureux et puissant, qui corrige les problèmes classiques de gestion mémoire du C tout en restant flexible et plus porteur sur le marché actuel.
- Le bon récit professionnel d’Aymerick est donc : backend Python aujourd’hui par pragmatisme de retour rapide à l’emploi, puis trajectoire volontaire vers Rust / systèmes / low-level à moyen terme par affinité profonde et ancienne pour cet univers.
