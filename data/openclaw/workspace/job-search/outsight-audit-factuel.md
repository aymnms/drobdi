## 2. PRODUIT & STACK

### Ce qu'ils font
- **Nature du produit:** Logiciel de traitement LiDAR temps réel (software ONLY)
- **Hardware:** Ils n'en font pas. Les capteurs viennent de vendors externes
- **Vendors de capteurs acceptés:** Ouster, Velodyne, Hesai, Robosense, Luminar (multi-vendor compatible)

### Produits nommés
- **Shift Perception:** Module de détection/classification/tracking d'objets en 3D temps réel
- **Shift Analytics:** Dashboard analytics, KPIs custom, alertes, historique
- **Augmented LiDAR Box (ALB):** Device edge computing avec software pré-intégré

### Stack Technique Confirmé
- **Backend:** Python 100% (confirmé par offre d'emploi)
- **Frontend:** Vue.js 3, Vuetify, Pinia
- **Infrastructure:** AWS, Docker, Terraform, Pulumi, CircleCI
- **Traitement:** Computer vision, 3D geometry, real-time processing
